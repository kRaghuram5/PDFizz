# PDFizz

A free, open-source PDF and document toolkit — convert, compress, edit, and transform PDFs with ease.

**Live App:** [pd-fizz.vercel.app](https://pd-fizz.vercel.app/)

---
## Features

| Operation | Description |
|-----------|-------------|
| PDF to Word | Convert PDF to editable Word document |
| PDF to Text | Extract plain text from a PDF |
| PDF to Images | Convert PDF pages to images (ZIP) |
| PDF to PowerPoint | Convert PDF pages to a PowerPoint presentation |
| Word to PDF | Convert Word documents to PDF |
| Excel to PDF | Convert Excel spreadsheets to PDF |
| PowerPoint to PDF | Convert PowerPoint presentations to PDF |
| Text to PDF | Convert a text file to PDF |
| Images to PDF | Combine multiple images into a single PDF |
| Merge PDFs | Combine multiple PDF files into one |
| Split PDF | Split a PDF into separate files |
| Compress PDF | Reduce PDF file size |
| Rotate PDF | Rotate pages in a PDF |
| Reverse PDF | Reverse the page order of a PDF |
| Remove Pages | Remove specific pages from a PDF |
| Extract Images | Extract all embedded images from a PDF (ZIP) |
| Add Page Numbers | Stamp page numbers onto a PDF |
| Watermark PDF | Add a text watermark to a PDF |
| Repair PDF | Attempt to repair a corrupted PDF |

---

## Tech Stack

- **Frontend:** React, deployed on Vercel
- **Backend:** Python (Flask), REST API
- **Storage:** Azure Blob Storage (for cloud deployments)
